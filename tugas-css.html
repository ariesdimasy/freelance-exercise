Penjelasan

  • Eksternal Style : yaitu pengaturan style css di luar file html. tujuannya biasanya dimana 1 file css tersebut untuk keseluruhan design dalam 1 project. Cara menghubungkan file css nya dengan file html nya adalah dengan menggunakan tag link
  • Internal Style : yaitu pengaturan style css di dalam file html. tujuannya untuk mengatur khusus 1 file html. pengaturann css nya berada dalam tag style dalam tag head
  • Inline Style : yaitu pengaturan style css di dalam 1 tag / 1 component html.

Diantara pengaturan css diatas, Tag inline yang paling kuat. Diikuti dengan Internal Style lalu Eksternal Style. itulah yang menjelaskan kenapa paragraph dengan class text-line berwarna kuning
