Penjelasan attribut

  1. width: untuk memberikan lebar pada komponen
  2. height: untuk memberikan tinggi pada komponen
  3. background-color: untuk memberikan background color pada komponen
  4. border: untuk memberikan border/pinggiran component. bisa di set ketebalan nya, tipe garis nya serta warnanya
  5. display: memberikan sifat dari element. apakah dia box ( yaitu menempatkan dirinya sebagai 1 row, tidak ada komponen lain di sampingnya )
    6. apakah dia inline ( yaitu menempatka nirinya sebagai text, ada component lain di sampingnya )
  6. margin-right : memberikan jarak ke sebelah kanan komponen
  7. color : memberikan warna pada huruf yg di dalam komponen
  8. border-radius : memberikan ketumpulan pada sudut component
  9. padding : memberikan jarak dari border terhadap isi dari element nya
  10. z-index: memberikan tingkat prioritas kemunculan component terhadap komponent yang lain
